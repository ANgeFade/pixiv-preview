# 🚀 快速安装指南

## 步骤1：打开Chrome扩展管理页面

在地址栏输入：
```
chrome://extensions/
```

或者通过菜单：**Chrome菜单 → 更多工具 → 扩展程序**

---

## 步骤2：启用开发者模式

在扩展管理页面的右上角，找到并**打开"开发者模式"**开关

![开发者模式](https://via.placeholder.com/600x100/667eea/ffffff?text=开启右上角的"开发者模式")

---

## 步骤3：加载扩展

1. 点击页面左上角的 **"加载已解压的扩展程序"** 按钮
2. 在文件选择对话框中，导航到：
   ```
   f:\code_program\pixiv-preview\extension
   ```
3. 选择该文件夹，点击 **"选择文件夹"**

---

## 步骤4：验证安装

✅ 扩展列表中出现 **"Pixiv 原图预览增强"**  
✅ 工具栏显示扩展图标（紫色渐变）  
✅ 状态显示为"已启用"

---

## 步骤5：开始使用

1. 访问 https://www.pixiv.net/
2. 鼠标悬停在任意作品缩略图上
3. 等待300毫秒
4. 🎉 高清原图预览出现！

---

## 📋 文件清单

确保 `extension` 文件夹包含以下文件：

```
extension/
├── ✅ manifest.json
├── ✅ background.js
├── ✅ content.js
├── ✅ styles.css
├── ✅ popup.html
├── ✅ popup.js
└── icons/
    ├── ✅ icon16.png
    ├── ✅ icon48.png
    └── ✅ icon128.png
```

---

## 🐛 遇到问题？

### 扩展无法加载
- 检查folder路径是否正确
- 确保所有文件都存在
- 查看错误信息提示

### 预览无法显示
1. 按 F12 打开开发者工具
2. 查看Console标签页
3. 检查是否有错误信息

### 图片加载失败
- 确保已登录Pixiv账号
- 检查网络连接
- 清除扩展缓存后重试

---

## 💡 提示

- **刷新页面**：修改扩展代码后，需要在扩展管理页面点击刷新按钮
- **查看日志**：在Pixiv页面按F12查看控制台输出的调试信息
- **清除缓存**：点击扩展图标 → 清除缓存按钮
